# 🎬 Watch Party

Синхронный просмотр видео для компании. Один ставит паузу — у всех пауза.

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить сервер
python server.py

# 3. Открыть в браузере
#    http://localhost:8000
```

## Использование

1. Введите ссылку на видеопоток (`.m3u8`, `.mp4`, `.webm` и др.)
2. Нажмите **Создать комнату** — получите уникальный URL
3. Поделитесь ссылкой с друзьями
4. Смотрите вместе: пауза/воспроизведение/перемотка синхронизируются для всех

## Поддерживаемые форматы

| Формат | Поддержка |
|--------|-----------|
| HLS (`.m3u8`) | ✅ через HLS.js |
| MP4 / WebM | ✅ нативный |
| DASH | ⚠️ частично |

## Архитектура

```
Browser ──WebSocket──► FastAPI Server
                           │
                    Комнаты (in-memory)
                    ├─ stream_url
                    ├─ is_playing
                    ├─ current_time
                    └─ connected WebSockets
```

## Запуск с HTTPS (для Интернета)

Для доступа из интернета используйте nginx как reverse proxy или ngrok:

```bash
# Быстро открыть для друзей через ngrok:
ngrok http 8000
```
