# 🎬 Watch Party

Синхронный просмотр видео для компании. Один ставит паузу — у всех пауза.

## Быстрый старт

```bash
# 1. Создать виртуальное окружение
python3 -m venv venv

source venv/bin/activate

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить сервер
python server.py

# 4. Открыть в браузере
#    http://localhost:8000
```

## Использование

1. Введите ссылку на видеопоток (`.m3u8`, `.mp4`, `.webm` и др.)
2. Нажмите **Создать комнату** — получите уникальный URL
3. Поделитесь ссылкой с друзьями
4. Смотрите вместе: пауза/воспроизведение/перемотка синхронизируются для всех

## Поддерживаемые форматы

| Формат | Поддержка |
|--------|-----------|
| HLS (`.m3u8`) | ✅ через HLS.js |
| MP4 / WebM | ✅ нативный |
| DASH | ⚠️ частично |

## Что бы быстро получать видео поток с сайтов

Необходимо установить расширение для бразуера

```
browser://extensions/
```

Загрузить распакованное расширение и указваем папку watch_party/watchparty-extension


## Архитектура

```
Browser ──WebSocket──► FastAPI Server
                           │
                    Комнаты (in-memory)
                    ├─ stream_url
                    ├─ is_playing
                    ├─ current_time
                    └─ connected WebSockets
```

## Запуск с HTTPS (для Интернета)

Для доступа из интернета используйте [tuna](https://tuna.am/) или [ngrok](https://ngrok.com/):

```bash
# Быстро открыть для друзей через ngrok:
ngrok http 8000

```

```bash
# Быстро открыть для друзей через tuna:
tuna http 8000

```
